Welcome to the SuiteCloud IDE

Getting Started

If you have not yet done so, run the Setup Wizard to create your master password and add an account:

⇒ Go to NetSuite > Setup Wizard.

If you are writing scripts for more than one account, you can add additional accounts:

⇒ Go to NetSuite > Accounts.

Next Steps

Once your Master Password and Accounts are set up, create a new NetSuite Project:

⇒ Go to File > New > NetSuite Project.

If you have existing SuiteScripts, download them from NetSuite to your local workspace:

⇒ Right-click on a project and go to NetSuite > Download File(s) to Project.

If you have existing custom fields or records in your NetSuite account, synchronize them with your local workspace to enable code completion for custom IDs:

⇒ Right-click on a project and go to NetSuite > Sync Custom Field and Record ID(s) from Account.

More Info

For more information, see the NetSuite Help and SuiteAnswers.
